# @pittorica/flex-react

The `Flex` component.

## Installation

```bash
npm install @pittorica/flex-react
```

## Usage

```jsx
import { Flex, Box } from '@pittorica/flex-react';

function MyLayout() {
  return (
    <Flex gap="4" align="center" justify="between">
      <Box>Item 1</Box>
      <Box>Item 2</Box>
    </Flex>
  );
}
```
