# pittorica

Core CSS design tokens and modern reset for the Pittorica UI library, inspired by Material Design 3 Expressive and Radix UI.

## Installation

```bash
npm install pittorica
```

## Usage

This package provides core CSS files that define design tokens and a modern CSS reset.
You can import them directly into your project:

```css
@import 'pittorica/reset';
@import 'pittorica/tokens';
/* Or import all core styles */
@import 'pittorica';
```
